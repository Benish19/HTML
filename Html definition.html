HTML

Introduction

HTML stands for HyperText Markup Language. It is the standard language used to create and design web pages. Every website you see on the internet is built using HTML (along with CSS and JavaScript).

What HTML Does

HTML is used to:
  • Structure content (text, images, links, videos)
  • Define headings, paragraphs, and layouts
  • Create forms and tables
    • Think of HTML as the skeleton of a website
